NutriTrack adalah Aplikasi Android modern untuk **melacak asupan kalori dan nutrisi harian** dengan mudah, dibangun sepenuhnya menggunakan **Jetpack Compose**.

## âœ¨ Fitur Utama
Aplikasi ini sedang dalam pengembangan aktif dan akan memiliki fitur-fitur berikut:

*   **Dashboard Utama:** Tampilan ringkasan kalori harian, progres, dan makanan yang telah dikonsumsi.
*   **Pemindaian Makanan:** Menggunakan kamera untuk mengidentifikasi makanan dari gambar dan menganalisis perkiraan nutrisinya.
*   **Pelacakan Makanan:** Menambahkan makanan (sarapan, makan siang, makan malam) secara manual.
*   **Autentikasi Pengguna:** Sistem login dan registrasi untuk menyimpan data pengguna secara personal.
*   **Wawasan & Tips:** Memberikan tips kesehatan berdasarkan data nutrisi pengguna.
*   **Splash Screen Animasi:** Memberikan kesan pertama yang menarik saat aplikasi dibuka.


## ğŸ› ï¸ Teknologi & Library yang Digunakan
Proyek ini dibangun menggunakan arsitektur modern Android dan memanfaatkan berbagai library dari ekosistem Jetpack.

*   **Bahasa Pemrograman:** [Kotlin](https://kotlinlang.org/)
*   **Toolkit UI:** [Jetpack Compose](https://developer.android.com/jetpack/compose) - untuk membangun UI native secara deklaratif.
*   **Navigasi:** [Compose Navigation](https://developer.android.com/jetpack/compose/navigation) - untuk mengelola alur layar di dalam aplikasi.
*   **Kamera:** [CameraX](https://developer.android.com/training/camerax) - untuk mengimplementasikan fitur pratinjau kamera dan pengambilan gambar.
*   **Machine Learning (on-device):** [Google ML Kit](https.developers.google.com/ml-kit)
    *   **Image Labeling:** Untuk mengidentifikasi objek makanan dari gambar.
*   **Asynchronous:** [Kotlin Coroutines](https://kotlinlang.org/docs/coroutines-overview.html) - untuk mengelola operasi background.
*   **Dependency Injection:** (Direkomendasikan: Hilt)
*   **Material Design:** [Material 3](https://m3.material.io/) - untuk komponen UI dan tema aplikasi.
